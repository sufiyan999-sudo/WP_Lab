Introduction

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages in the world.

Python's design philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Python is dynamically typed and garbage-collected. It supports multiple programming paradigms, including structured, object-oriented, and functional programming. Python is often described as a "batteries included" language due to its comprehensive standard library.

  • Easy to learn and use
  • Extensive standard library and third-party packages
  • Cross-platform compatibility

Getting Started

To start programming in Python, you first need to install Python on your computer. You can download the latest version from the official Python website at python.org. Python is available for Windows, macOS, and Linux operating systems.

Once installed, you can verify your installation by opening a command prompt or terminal and typing python --version. This will display the installed Python version.

Your first Python program is traditionally a "Hello World" script. Here's how you can create one:

print("Hello, World!")

You can run this program by saving it to a file with a .py extension and executing it with the command python filename.py.

Variables and Data Types

In Python, variables are created when you assign a value to them. You don't need to declare the variable type explicitly, as Python is dynamically typed. The interpreter automatically determines the type based on the value assigned.

Here are some common data types in Python:

  • Integers: Whole numbers like 42 or -17
  • Floats: Decimal numbers like 3.14 or -0.5
  • Strings: Text data enclosed in quotes like "Hello" or 'World'
  • Booleans: True or False values
name = "Alice"
age = 30
height = 5.6
is_student = False

Control Flow

Control flow statements allow you to control the execution of your code based on certain conditions. Python uses indentation to define code blocks, which makes the code more readable.

The if statement is used for conditional execution. Here's an example:

if age >= 18:
    print("You are an adult")
elif age >= 13:
    print("You are a teenager")
else:
    print("You are a child")

Python also provides loop statements like for and while for repetitive tasks. The for loop is commonly used to iterate over sequences:

for i in range(5):
    print(i)

Functions

Functions are reusable blocks of code that perform specific tasks. In Python, you define a function using the def keyword, followed by the function name and parentheses.

Functions can accept parameters and return values. Here's a simple example:

def greet(name):
    return f"Hello, {name}!"

message = greet("Alice")
print(message)

Functions help organize code, make it more maintainable, and allow code reuse. You can also define functions with default parameter values and variable-length argument lists.

  • Functions improve code organization
  • They promote code reusability

Lists and Dictionaries

Lists and dictionaries are two of the most commonly used data structures in Python. Lists are ordered collections of items that can be of any type, while dictionaries store key-value pairs.

Here's an example of a list:

fruits = ["apple", "banana", "cherry"]
fruits.append("orange")
print(fruits[0])  # Output: apple

And here's an example of a dictionary:

person = {
    "name": "Alice",
    "age": 30,
    "city": "New York"
}
print(person["name"])  # Output: Alice

Both lists and dictionaries are mutable, meaning you can modify them after creation. They are essential for organizing and managing data in Python programs.
